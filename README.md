# lorchard 
基于小程序开发的b2c商城,

# 搭建环境
>> Jdk 1.7
>
>> Maven
>
>> Nysql 5.6
>
>> Tomcat 7.0
>
>> Redis
>
>> Rabbit-mq
>

# 技术栈
## 后台技术
>> spring 4.3 bean 管理容器
>
>> mybatis 3.2 数据持久化
>
>> druib 数据库连接池
>
>> redis 内存数据库
>
>> shiro 权限管理框架
>
>> rabbit-mq 消息队列
>
>> jsp 界面渲染
>
>> netty 搭建 websocket
>
## 前台技术
>> layer
>
>> jquery
>
>> amazeUI
>
>> mina 小程序 
>
## 模块
>> common 公共模块
>
>> api 小程序端api
>
>> applet 小程序客户端
>
>> merchant 商家端
>  
>> mq 消息队列
>
>> shiro 权限管理
>
>> admin 商家平台端
>
>> websocket websocket推送
>
